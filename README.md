# Network Simplex

## Build & Test
```
mkdir build 
cd build
cmake ..
make
sh ../script/unittest.sh
```

## Usage
```
./cnet $FILE_NWK [$DEBUG]
```

## Reference

1. Florian, M., & Lebeuf, D.. (1997). An efficient implementation of the network simplex method. Network Optimization. Springer Berlin Heidelberg.
